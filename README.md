# Minecraft Manager

A free desktop app to host, manage, and share Minecraft servers — no terminal needed.

## Features

- **One-click server creation** — Paper, Purpur, Fabric, Forge, NeoForge, Vanilla
- **Console** — Live output with filters, level filter, crash report viewer, save log
- **Players** — Online players, ops, whitelist, banned players, banned IPs, alt detection
- **World Manager** — View world sizes, set world border, reset worlds
- **Paper Settings** — Exploit fixes, gameplay toggles, Anti-Xray configuration
- **Backups** — Scheduled and manual backups with one-click restore
- **Plugins / Mods** — Browse and install from Modrinth
- **TPS Monitor** — Live TPS badge for Paper/Purpur servers
- **Network** — UPnP port forwarding, DuckDNS, Playit.gg tunneling
- **Version Updater** — One-click Paper/Purpur updates
- **Import / Export** — Import existing server folders or `.mcm` config files
- **MOTD Preview** — Live color preview of your server description
